1-Purpose:
Ternary Operator: It is used to evaluate a condition and return a value based on whether the condition is true or false. It is a shorthand way of writing an if-else statement that returns a value.
if-else statements: They are used to execute a block of code if a specified condition is true. If the condition is false, another block of code can be executed.
Switch statements: They are used to execute one block of code from multiple options based on the value of an expression.

2-Syntax:
Ternary Operator: (condition) ? expression1 : expression2

3-Usage:
Ternary Operator: It is commonly used for assigning values to variables or returning values from functions based on a condition.
if-else statements: They are used when you have to evaluate multiple conditions or when the condition is more complex.
Switch statements: They are used when there are multiple cases to be
evaluated against a single expression. Switch statements are particularly useful when dealing with enum types or when there are many possible values to check against.

4-Number of Conditions:
Ternary Operator: It evaluates only one condition.
if-else statements: They can handle multiple conditions by using else-if clauses.
Switch statements: They can handle multiple conditions, but each case must evaluate to a constant value.

5-Execution Flow:
Ternary Operator: It returns one of two values based on the result of a condition.
if-else statements: They allow for more complex branching of code execution based on various conditions.
Switch statements: They provide a way to select from multiple options based on the value of an expression.

6-Readability:
Ternary Operator: It can sometimes make code harder to read, especially for complex conditions.
if-else statements: They are more readable, especially for longer conditions and multiple branches of code.
Switch statements: They can be more readable when there are many options to choose from, but they may become complex if there are too many cases.
