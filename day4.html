1-Number:
The number data type in JavaScript represents both integer and floating-point numbers.
It follows the IEEE 754 standard for floating-point arithmetic. Numbers can represent values within the range
of approximately ±5 x 10^-324 to ±1.7976931348623157 x 10^308, including both integer and fractional parts.

2-BigInt:
The BigInt data type, introduced in ECMAScript 2020 (ES11), is used to represent arbitrary-precision integers.
Unlike regular numbers, BigInts can represent integers with arbitrary precision, limited only by available memory.
BigInts are denoted by appending n to the end of an integer literal or by using the BigInt() constructor.
