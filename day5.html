DIFFERENCIATE BTW COMPARISONS AND LOGICAL OPEREATORS
Comparison Operators: Comparison operators are symbols used to compare two values or expressions. They evaluate to a Boolean value
(true or false) depending on the relationship between the operands. Common comparison operators include:
1-Equal to (==): Checks if two values are equal.
2-Not equal to (!=): Checks if two values are not equal.
3-Greater than (>): Checks if the left operand is greater than the right operand.
4-Less than (<): Checks if the left operand is less than the right operand.
5-Greater than or equal to (>=): Checks if the left operand is greater than or equal to the right operand.
6-Less than or equal to (<=): Checks if the left operand is less than or equal to the right operand.
7-Example (Python): python
Copy code
x = 5
y = 10
print(x == y) # False
print(x < y) # True
print(x >= y) # False
Logical Comparison: Logical comparisons involve using logical operators to combine multiple comparison expressions.
These operators include:
1-AND (and): Returns true if both operands are true.
2-OR (or): Returns true if at least one of the operands is true.
3-NOT (not): Returns the opposite of the operand's Boolean value.
Example (Python): python
Copy code
x = 5
y = 10
z = 3
# Logical AND
print(x < y and y < z) # False
# Logical OR
print(x < y or y < z) # True
# Logical NOT
print(not (x < y)) # False
In summary, comparison operators are used to compare individual values or expressions,
while logical comparisons involve combining multiple comparison expressions to form more complex conditions.
